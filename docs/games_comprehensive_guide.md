## Crash Game（崩盘游戏）

> 也称为：飞机游戏 (Aviator)、火箭游戏、Bustabit

### 📖 基本概念

Crash Game 是一种以**倍数增长和即时决策**为核心的实时博彩游戏。游戏的核心张力在于：贪婪与风险的博弈——等待更高倍数意味着更大收益，但也可能瞬间失去一切。

### 🎯 玩法规则

```
游戏流程图：

  下注阶段         倍数增长阶段              结算阶段
  [10秒倒计时]  →  [1.00x → 1.50x → 3.00x → ?]  →  [结算]
       ↓                    ↓                        ↓
    玩家下注           玩家可随时提现            崩盘/提现成功
```

**详细步骤：**

| 阶段 | 描述 | 玩家操作 |
|------|------|----------|
| 1️⃣ 下注 | 游戏开始前的等待期（通常10-15秒） | 设置下注金额，可设置自动提现倍数 |
| 2️⃣ 起飞 | 倍数从1.00x开始持续增长 | 观察倍数，决定提现时机 |
| 3️⃣ 提现 | 点击按钮锁定当前倍数 | 收益 = 下注额 × 当前倍数 |
| 4️⃣ 崩盘 | 倍数在随机时刻突然停止 | 未提现则失去全部下注 |

### ⚡ 特殊功能

| 功能 | 说明 |
|------|------|
| **自动提现（Auto Cash Out）** | 预设目标倍数，自动提现 |
| **双倍下注** | 同时进行两笔独立下注，分别控制 |
| **历史记录** | 查看往期崩盘倍数，分析趋势（注：无预测价值） |
| **Provably Fair** | 可验证公平算法，玩家可自行验证结果真实性 |

## Slots（老虎机）

> 也称为：角子机、拉霸机、Slot Machine

### 📖 基本概念

Slots 是最经典的赌场游戏类型，起源于19世纪末的机械设备。现代视频老虎机结合了丰富的主题、动画效果和多样化的奖励机制，是在线赌场中数量最多、玩家最广的游戏类型。

核心原理：通过随机数生成器（RNG）决定符号组合，根据赔付表确定输赢。

### 🎯 玩法规则

```
标准5卷轴老虎机布局：

    列1   列2   列3   列4   列5
  ┌─────┬─────┬─────┬─────┬─────┐
  │ 🍒  │ 🍋  │ 🍇  │ 🍊  │ 🔔  │  行1
  ├─────┼─────┼─────┼─────┼─────┤
  │ 💎  │ 🍒  │ 🍋  │ 🍇  │ 🍊  │  行2 ← 中心赔付线
  ├─────┼─────┼─────┼─────┼─────┤
  │ 7️⃣  │ 💎  │ 🍒  │ 🍋  │ 🍇  │  行3
  └─────┴─────┴─────┴─────┴─────┘
    R1    R2    R3    R4    R5
         (5个卷轴/Reels)
```

**游戏步骤：**

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1️⃣ | 选择下注额 | 设置单线注额和赔付线数 |
| 2️⃣ | 点击旋转 | Spin按钮启动游戏 |
| 3️⃣ | 卷轴停止 | 显示随机符号组合 |
| 4️⃣ | 判定结果 | 系统检查赔付线上的符号组合 |
| 5️⃣ | 结算奖金 | 根据赔付表计算并发放奖金 |
